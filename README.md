# You Type What You Are 🐱🚗☕
# ALL CODEBASE AI GENERATED, DO NOT THROW TOMATOS TO ME PLEASE !!! 
A tiny, funny, and slightly judgmental typing experiment.

If you type **`cat`** — congrats, you *are* a cat 🐱  
If you type **`car`** — you’re going way too fast 🚗💨  

Slow down.  
Breathe.  
Take a tea break.  
Enough coffee already ☕😌

---

## 🧠 The Idea

Words say more than we think.

This project plays with the idea that **what you type reflects your current state of mind**:

- 🐱 **cat** → calm, cozy, vibing  
- 🚗 **car** → rushing, stressed, caffeine-powered  

When the system detects “fast” or stressed words, it gently (or not so gently) reminds you to slow down and take a break.

Because honestly…  
**No one needs their 5th coffee today.**

---

## ✨ Features

- ⌨️ Detects what the user types
- 🧩 Interprets words as *mood indicators*
- ☕ Suggests breaks instead of burnout
- 😼 Zero productivity pressure
- 💚 Maximum vibes

## 🎯 Why This Exists

- To remind developers (and humans) to **slow down**
- To turn typing into a tiny moment of self-awareness
- To make burnout prevention a little more fun

This is not a productivity tool.  
This is an **anti-overwork reminder disguised as a joke**.
